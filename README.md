# city-love

Cinematic romantic messages across Iraq. Send a message from one province to another, and the receiver sees a beautiful animated journey.

## Getting Started

1. Set up Supabase:
   - Create a Supabase project
   - Run the SQL in `supabase/schema.sql`
   - Copy your URL and anon key to `.env.local`

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the dev server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000)
